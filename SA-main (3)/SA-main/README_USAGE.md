# 匿名論壇系統 - 安裝並開啟系統說明

1.開啟SA-main(3)後
輸入cd SA-main
輸入cd project
輸入cd backend
輸入java -jar target\demo-0.0.1-SNAPSHOT.jar --spring.profiles.active=h2

2.打開網站

在任何瀏覽器的位址列輸入或直接點以下按鈕：
http://localhost:8080/main.html

3.測試

3-1.請先註冊一般使用者(註冊成功後即可用該帳號登入)
在「註冊」頁面填入以下資訊：
真實姓名(例：王小明)
Gmail 帳號例：user@gmail.com）
密碼：至少 8 位英數字混合（例：Password123）
確認密碼：重新輸入密碼

3-2.用一般使用者登入

3-3.發佈貼文

3-4.查看文章

3-5.點顯示真實姓名按鈕

3-6.登出後管理員登入
管理員帳號(登入後可以查看所有文章的發文者真實姓名)
Email:admin@gmail.com
密碼:adminpassword(網站上目前有預設一串輸入行，需要手動改成此密碼，才可登入到管理員帳號)

3-7.管理員點顯示真實姓名按鈕

4.網頁主要功能

登入/註冊：使用 Gmail 帳號註冊與登入
文章列表：瀏覽所有發布的文章，以及搜尋功能
發布文章：登入後可發布新文章（只有登入使用者可見發布按鈕）
文章詳情：點擊文章標題查看完整內容與留言
留言功能：登入後可在文章底部留言
查看真實姓名：管理員可點擊「查看真實姓名」按鈕查看發文者與留言者的真實身份

5.常見問題

登入時顯示「帳號或密碼錯誤」，確認以下幾點：
Email 格式正確（必須是 Gmail）
密碼輸入無誤（注意大小寫）
帳號確實已註冊

註冊時顯示「Email 可能已存在」
該 Email 已被註冊過，請使用不同的 Email。

6.環境

後端：Java Spring Boot 3.2.1
前端：靜態 HTML + JavaScript
資料庫：H2、mysql(用來測試java是否可以正確運行)
